이 폴더에 사진을 아래 파일명으로 넣으면 홈페이지에 자동으로 표시됩니다.

  poster.jpg     → ★ 메인 포스터 (맨 위 히어로에 크게 표시) ※ 가장 중요
  cover.jpg      → 펜션 전경 (갤러리 대표)
  living.jpg     → 1층 폴딩도어 거실 (미롱가 공간)
  yard.jpg       → 노을빛 마당 (BBQ)
  terrace.jpg    → 야외 테라스 테이블 (아침식사)
  floorplan.jpg  → 펜션 도면 (방 배정)

★ poster.jpg 가 없으면 텍스트 자리표시 히어로가 대신 보입니다.
   채팅에 올린 포스터 이미지를 이 폴더에 'poster.jpg' 이름으로 저장하세요.

* 파일 형식은 .jpg 권장. (.png 라면 index.html 의 파일명도 .png 로 맞춰주세요)
